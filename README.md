# **FinnaCloud** (TheFinnaCompany Ltd)

FinnaCloud is a small, engineering-driven cloud and software venture based in the United Kingdom.

We offer **consultancy**, **SaaS platforms**, and **cloud solutions** built on top of established hyperscale providers (AWS, GCP, Azure, etc.), enabling flexible and high-performance systems **without vendor lock-in**.

Inclusivity and diverse perspectives are core to our culture. FinnaCloud is built on the belief that better technology comes from broader viewpoints and collaborative engineering.

---

# **Open Source**

## **Open-Source Cloud & Developer Tools**

We believe that cloud infrastructure and developer tooling should be **open**, transparent, and accessible.  
FinnaCloud is committed to releasing selected internal tools, libraries, and services as **Open-Source Software (OSS)**, empowering others to learn from and build upon our work.

Contributions of all kinds are welcome.

# **Contributing**

## **Code of Conduct**

All FinnaCloud repositories follow a strict Code of Conduct to ensure a welcoming, respectful, and inclusive environment. Professional collaboration and respectful communication are expected at all times.

## **Types of Contributions**

We welcome contributions such as:

- Code  
- Documentation  
- Bug reports  
- Feature requests  
- Platform or ecosystem improvements  

Every contribution helps strengthen the project.

## **Contribution Guidelines**

Before contributing, please review our contribution guidelines:  
https://github.com/FinnaCloud/FinnaCloud/contributing.md

We encourage:

- Clean, well-documented code  
- Detailed issue reports  
- Clear, structured pull requests  
- Open and constructive collaboration  

## **Getting Started**

1. Fork the repository  
2. Create a new branch  
3. Make your changes  
4. Open a pull request  

Our team will review your submission as soon as possible.
